## Run Locally

Clone the project

```bash
  git clone https://github.com/lobnor2/Modern-UI-UX-Tailwind
```

Go to the project directory

```bash
  cd Modern-UI-UX-Tailwind
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
